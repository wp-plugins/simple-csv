== Description ==

The Simple CSV plugin allows to export posts from any category to a csv file. Simple CSV is a simple and effective exporter and a perfect solution for those people who have no time for a bunch of settings. Just a couple of clicks and you`ll see a csv file on your Desktop.

= Features =

* Exports all post fields
* Exports post meta
* Export Posts for specific date range


= Simple CSV =
  Simple CSV is a fully functional but limited version of our full [Simple CSV plugin](http://cbfreeman.com/downloads/simple-csv/).

  Our full version includes many more features:

* Select one or multiple categories of post to a csv file

* Select All - This feature allows you to print all posts in every category to a csv file.

== Installation ==

For an automatic installation through WordPress:

1. Go to the 'Add New' plugins screen in your WordPress admin area
1. Search for 'Simple CSV'
1. Click 'Install Now' and activate the plugin
1. Go the 'Users' menu, under 'Simple CSV'


For a manual installation via FTP:

1. Upload the `simple-csv` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' screen in your WordPress admin area
1. Go the 'Users' menu, under 'Simple CSV'


To upload the plugin through WordPress, instead of FTP:

1. Upload the downloaded zip file on the 'Add New' plugins screen (see the 'Upload' tab) in your WordPress admin area and activate.
1. Go the 'Users' menu, under 'Simple CSV'

== Frequently Asked Questions ==

= How to use? =

Click on the 'Simple CSV' link in the 'Users' menu, choose the category and the date range, then click 'Export Posts'. That's all!
